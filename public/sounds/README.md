# UI sounds

Four short cues, played by `src/client/sound.ts`:

- `click.ogg` — a button press (MPC vote, risk vote, ready, minigame click/rescue).
- `success.ogg` — a round is won.
- `failure.ogg` — a round is lost.
- `tick.ogg` — the last 5 seconds of any countdown.

Source: [Kenney — Interface Sounds](https://kenney.nl/assets/interface-sounds) (CC0),
renamed from `click_001`, `confirmation_001`, `error_001`, `tick_001`.
